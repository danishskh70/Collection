    /* 
    Collection -->
    A Collection represent Single Unit of Objects
    Framework -->
    To Specific task , we require specific class, interface, functions
    So we put all Pre-Written Codes  like class,function,interface inside one place called Framework
    Framework is body contain pre written code in whichwe can add our code to overcome the problem of reusability
    
    Modules -> atomic task /set of method ar function ready to use anywhere
    packages -> collection of modules /
    library -> collection of packages
    Framework -> collection of Library + gives Architectural Style i.e call, define,declare
    
    
    Framework --> Library --> Packages -->Modules
    
    
    */
